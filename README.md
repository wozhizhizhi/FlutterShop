# FlutterShop

Flutter的移动电商项目

## 用到的组件

- dio:用于向后端接口作HTTP请求数据
- flutter_swiper: 轮播插件，制作了商城首页的轮播效果
- flutter_screenutil:用于不同屏幕的适配
- url_launcher:用于打开网页和实现原生电话的拨打
- flutter_easyrefresh：下拉刷新或者上拉加载插件。
- provide：谷歌最新推出的Flutter状态管理插件。
- fluttertoast：Toast轻提示插件。
